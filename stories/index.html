📁 案件檔案　CASE FILE #001 · 飲料容器類

信天翁的胃裡有一支打火機

地點太平洋中途島 (Midway)
受害者100 萬隻信天翁雛鳥
兇器瓶蓋、打火機、玩具兵
加害者陸地上的人類

中途島，太平洋中央，距離夏威夷 2000 公里。 島上沒有人，但有 100 萬隻信天翁築巢。

2009 年，攝影師 Chris Jordan 來這裡拍照。他看到的是——
一隻一隻死掉的信天翁雛鳥，肚子被剖開後，全是塑膠。

打火機、瓶蓋、牙刷、玩具兵、髮夾、原子筆。 全部是母信天翁從海上叼回來餵小鳥的—— 因為塑膠在陽光下會發亮，看起來像魚。

小鳥吃了塑膠胃裡塞滿、但沒營養。活活餓死。

🤯

這些塑膠，都來自陸地。 意思是——你今天在台南丟的瓶蓋，有可能真的會跑進中途島某隻雛鳥的胃。

來源：Chris Jordan《Midway》紀錄片（2018）、USFWS 中途島野生動物保護區。

📁 案件檔案　CASE FILE #002 · 食物包裝類

一根吸管，改變了一個國家的政策

地點哥斯大黎加海岸
日期2015 年
受害者一隻欖蠵龜（公）
兇器10 公分塑膠吸管

2015 年，海洋生物學家 Christine Figgener 在做欖蠵龜研究。 她發現一隻公龜鼻子裡卡著什麼東西。

團隊用鉗子一點一點拔。 海龜不停地痛苦哀鳴，鼻子流血。

整整 8 分鐘，他們拉出一根10 公分長的塑膠吸管。 深深塞在海龜鼻腔。

影響
1 億+
YouTube 影片觀看次數
這 8 分鐘改變了世界——後來歐盟、台灣、日本、加拿大、新加坡都禁了一次性塑膠吸管。

台灣2019 年內用禁吸管，主因之一就是這支影片。 一根吸管，引發一波全球政策。

來源：Christine Figgener (Texas A&M University) YouTube 公開影片，搜「sea turtle straw」可看。

📁 案件檔案　CASE FILE #003 · 漁業用具類

幽靈漁網——還在抓魚的死亡陷阱

地點夏威夷群島
瀕危等級世界僅剩 1500 隻
受害者夏威夷僧海豹（與其他海洋動物）
兇器被棄置的漁網（ghost net）

想像有一張漁網，沒有人去收。 它沉在海裡、漂在表面、卡在礁石—— 但它還在繼續抓魚。

海龜會被纏住、海豚會被纏住、海豹也會被纏住。 有的脫得開，有的脫不開——脫不開的就溺死。

這就是「幽靈漁網（ghost net）」。 全球海洋裡有64 萬噸，相當於 5 萬輛公車的重量。

這張照片你應該看過

一隻夏威夷僧海豹，脖子被廢棄漁網纏住，被海浪沖到沙灘上喘氣。 （圖見「漁業用具」分類頁）

全世界夏威夷僧海豹只剩 1500 隻。 少一隻，就是少 0.07%——這個物種隨時可能滅絕。

NOAA 漁業署每年從夏威夷海域移除 50 噸幽靈漁網。 但這只是冰山一角——世界海洋裡還有 64 萬噸。

來源：NOAA Fisheries、World Animal Protection《Ghost Gear》報告（2018）。

📁 案件檔案　CASE FILE #004 · 本地特殊案件

★ 安平蚵棚——你家旁邊的故事

地點臺南安平 · 漁光島月牙灣
關係人5 年級兩班、社大志工、蚵農
兇手保麗龍浮具碎片
狀態★ 進行中（你會去現場）

這個案件就在你家附近。

台南、嘉義、雲林是全台最大的蚵類養殖區。 蚵農在海裡搭蚵棚——竹竿+保麗龍浮具+繩子，蚵就掛在繩子上長 6 個月。

但每年颱風一來——

  • 蚵棚被打壞
  • 保麗龍碎成沙子大小的小白點
  • 漂上漁光島月牙灣
  • 混在沙裡，永遠清不乾淨
不會分解
200+ 年
保麗龍碎片在沙裡的時間
而且它不會「分解」——只會碎得更小。
最後跑進魚的肚子，跑進蚵的肚子——而蚵就是養來給人吃的。

在地英雄：台南社大海廢社

台南社區大學從 2005 年開始監測月牙灣，20 年沒停。 每月一次去撿、紀錄、比對。

兩週後，我們會跟社大志工阿凱合作。 我們撿的數據，會加進他們 20 年的資料庫。

🦸

你不只是去玩沙——你會在做科學家在做的事。 你的資料，會跟全國、全球的志工資料合在一起。

來源：台南社區大學環境組（beach.tncomu.tw）。
